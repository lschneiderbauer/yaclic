yaclic by vootey

Yet Another Cli Calculator

Yaclic is a simple and intuitive cli-calculator, where symbols can be as easily used as numbers. Cause of writing such a calculator again is, that there exists just the "super-dumb-simple" ones and the "heavily-armed" ones (or at least I didn't find anything between). So that's my solution (and there's work in progress). It constists more or less of an extended ruby-shell.

Dependencies

Ruby 1.8

Install

You can install Yaclic as Ruby-Gem. Just pull the sources and do gem build ./yaclic.gemspec gem install yaclic-{v}.gem where v is the current version of yaclic.
